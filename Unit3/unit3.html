3.1 Conceptual Modelling

Conceptual modelling helps designers simulate and understand their designs before production, considering functionality, behavior, appearance, and user interaction.

Advantages

  • Client input opportunities
  • Cost-effective exploration
  • Reduced development costs
  • Proof of concept testing
  • Function communication
  • Aesthetic visualization
  • Material testing
  • Stakeholder feedback

Disadvantages

  • May miss important details
  • Not fully representative
  • Material limitations
  • Scale proportion issues
  • Limited durability

3.2 Graphical Modelling

Graphical modelling encompasses both 2D and 3D representations, each serving different purposes in the design process.

Orthographic Drawing

Shows detailed side views without perspective.

  • No vanishing points
  • Accurate form representation
  • Manufacturing specifications
  • Dimensional accuracy

Isometric Drawing

30/90/30 degree grid representation.

  • No vanishing points
  • Overall form communication
  • Assembly visualization
  • Part relationships

Perspective Drawing

Single viewpoint representation with depth.

  • Uses foreshortening
  • Has vanishing points
  • Realistic appearance
  • Early planning visualization

3.3 Physical Modelling

Physical models provide tangible information about design aspects including ergonomics, structure, and aesthetics.

Scale Models

  • Smaller/larger physical copies
  • Architectural visualization
  • Testing capabilities
  • Form communication

Aesthetic Models

  • Look and feel evaluation
  • User appeal testing
  • Material properties
  • Surface qualities

Prototypes

  • Low-fidelity concepts
  • Mid-fidelity functions
  • High-fidelity testing
  • Specification refinement

3.4 CAD (Computer-Aided Design)

CAD software enables designers to create precise digital models and documentation for their designs.

2D Software

  • Adobe Illustrator
  • Adobe Photoshop
  • Technical drawings
  • Design documentation

3D Software

  • Autodesk Fusion 360
  • SketchUp
  • Dimensional modeling
  • Material properties

Rendering Software

  • Blender
  • Maya
  • 3D Studio Max
  • Realistic visualization
